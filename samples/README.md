# Sample Input Documents

This folder is the canonical place for shareable sample inputs used during
submission review and demos.

The repository includes these synthetic PDFs:

```text
samples/demo_invoice.pdf
samples/sample_receipt.pdf
samples/sample_resume.pdf
samples/sample_identity_card.pdf
samples/sample_bank_statement.pdf
samples/sample_multilingual_invoice.pdf
samples/sample_receipt_scanned.png
samples/sample_bank_statement_photo.png
samples/sample_identity_card_photo.png
samples/sample_handwritten_note.png
```

Use these files to verify upload, preprocessing, OCR, extraction, validation,
classification, and Q&A workflows without exposing private documents.

`sample_handwritten_note.png` is a synthetic handwriting-style image intended
for a basic open-source OCR handwriting check. It is not private user data.
